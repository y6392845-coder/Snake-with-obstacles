Описание проекта:
Программный продукт "Змейка с препятствиями" - это классическая аркадная игра с модификациями, реализованная на языке Python с использованием библиотеки Tkinter для графического интерфейса.

-----------------------------------------------------------------------------------------------------------------------------

Главная идея проекта:
Создать увлекательную игру "Змейка" с дополнительными функциями и усложнениями, которые делают игровой процесс более интересным и сложным по мере прогресса игрока. Проект демонстрирует применение принципов объектно-ориентированного программирования и работу с графическим интерфейсом.

-----------------------------------------------------------------------------------------------------------------------------

Основные особенности игры:
Динамическая сложность: С увеличением уровня появляются новые препятствия

Система уровней: Уровень повышается каждые 50 очков

Пауза и управление: Возможность приостановить игру и несколько вариантов управления

Визуальные эффекты: Разные цвета для головы и тела змейки, препятствий и еды

Экран завершения игры: Информативный экран с итоговыми результатами

-----------------------------------------------------------------------------------------------------------------------------


Для создания приложения используется:
tkinter:

Основная библиотека для создания графического интерфейса

Используется для создания окна игры, кнопок, меток и холста для рисования

Обеспечивает обработку событий клавиатуры

random:

Используется для генерации случайных положений еды и препятствий

Обеспечивает случайность игрового процесса

-----------------------------------------------------------------------------------------------------------------------------

Ключевые компоненты приложения:
1. Графический интерфейс:
Игровое поле (Canvas) для отрисовки игровых объектов

Панель информации с текущим счетом и уровнем

Кнопка для начала новой игры

Инструкция по управлению

2. Игровая логика:
Управление змейкой с помощью клавиш WASD или стрелок

Система столкновений (со стенами, собой, препятствиями)

Система подсчета очков и уровней

Генерация новых препятствий при повышении уровня

3. Визуальные элементы:
Змейка с выделенной головой (зеленый цвет) и телом (темно-зеленый)

Препятствия (серые квадраты)

Еда (красные круги)

Анимированное движение объектов

-----------------------------------------------------------------------------------------------------------------------------

Архитектура проекта:
Класс SnakeGame - основной класс игры, содержащий:

Инициализацию графического интерфейса

Логику игры и обработку событий

Методы отрисовки игровых объектов

Игровой цикл и механику обновления
